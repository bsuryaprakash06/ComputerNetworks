# Computer Networks Lab and Protocol Implementations

This repository contains practical implementations of fundamental **Computer Networks concepts**, including **data link layer protocols, transport layer communication using sockets, and network utilities**.

The repository is structured to provide hands-on understanding of how network protocols work internally and how communication is established between systems.

---

## Repository Structure

### Data Link Layer Protocols

* **2a_Stop_and_Wait_Protocol**

  * Implementation of Stop-and-Wait ARQ
  * Reliable data transmission using acknowledgments

* **2b_SLIDING_WINDOW_PROTOCOL**

  * Sliding Window Protocol implementation
  * Flow control and efficient data transfer

* **2c_ARP_RARP_PROTOCOLS**

  * Address Resolution Protocol (ARP)
  * Reverse ARP (RARP)
  * Mapping between IP and MAC addresses

---

### Socket Programming

* **3a_Sockets_Creation_for_Echo_Client_and_Echo_Server**

  * Basic TCP socket creation
  * Echo client-server communication

* **3b_CHAT_USING_TCP_SOCKETS**

  * Two-way communication using TCP sockets
  * Real-time chat application

* **3c_FILE_TRANSFER_USING_TCP_SOCKETS**

  * File transfer between client and server
  * Reliable transmission using TCP

---

### Networking Utilities

* **4_Execution_of_NetworkCommands**

  * Execution of common network commands
  * Examples: ping, traceroute, netstat

---

### HTTP and Application Layer

* **5a_Create_Socket_for_HTTP_for_webpage_upload**

  * HTTP communication using sockets
  * Basic web client/server interaction

---

### Practice Modules

* **ChatStudy**

  * Additional experiments with chat applications

* **SocketStudy**

  * Additional socket programming exercises
